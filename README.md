# PERCH

**Stay perched on reality while agents work.**

Minimal terminal file viewer for monitoring changes in any git directory.

## Install

```
go install github.com/kateleext/perch@latest
```

## Usage

```
perch [directory]
```

## Keybindings

| Key | Action |
|-----|--------|
| `↑↓` | Navigate files |
| `j/k` | Scroll preview |
| `g/G` | Top/bottom |
| `q` | Quit |
| `shift` + select | Copy text |
